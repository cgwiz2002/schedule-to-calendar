Schedule-to-Calendar

Personal Automation Tool

This is a private script that helps me (Peter Lofaro) automatically process Excel/CSV schedules and add them to my Google Calendar.

What it does

  • Reads engineering job schedules (Sainsbury’s, Argos, etc.)
  • Creates calendar events with full details
  • Sends notifications via Gmail / Telegram
  • Runs on my personal machine
